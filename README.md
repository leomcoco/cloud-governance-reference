# Cloud Governance Reference

This repository provides a vendor-neutral reference for cloud governance
principles, patterns and decision-making across multiple cloud providers.

It focuses on the "why" and "what" of governance, serving as a conceptual
foundation that can be applied to Azure, AWS and GCP.

## Scope
- Cloud governance principles
- Guardrails and compliance models
- Organizational structures
- Policy-driven governance concepts
- Architectural decision records (ADR)

## Cloud coverage
- Azure
- AWS
- Google Cloud Platform (GCP)

## Audience
Cloud architects, platform engineers and technical leaders
seeking structured and scalable governance guidance.

## Relationship with other repositories
Practical implementations of the concepts described here
are documented in cloud-specific repositories
(e.g., `azure-infrastructure-labs`).
